# 乡村振兴数据可视化大屏

## 项目背景
本项目围绕“乡村振兴”主题，构建一个集产业、人口、生态与非遗信息于一体的数据可视化大屏，用于展示乡村综合发展情况。

## 技术栈
- Vue 3
- Vite
- Vue Router
- ECharts

## 核心功能
- 3 个页面路由：首页、产业分析、生态环境。
- 首页左中右三栏大屏布局，包含地图模块与 6 类图表：
  - 折线图
  - 柱状图
  - 饼图
  - 雷达图
  - 仪表盘
  - 热力图
- 图表支持 Tooltip 提示，地图点击可联动柱状图数据。
- 动态数字翻牌显示“乡村年总产值”。
- 统一深蓝科技感主题与非遗装饰元素。
- 适配 1920x1080 与 1366x768，无滚动条。

## 运行步骤
```bash
npm install
npm run dev
```

## 数据来源说明
项目数据为本地静态模拟数据，存放在 `src/assets/json/data.json`，用于满足教学与演示场景下的可视化展示需求。
