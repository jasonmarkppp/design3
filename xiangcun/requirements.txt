第一阶段：项目初始化（工程化与基础架构）
目标： 满足考核要求 1（Vue 3 + ECharts + Router）和要求 7（结构清晰）。

提示词：
我正在按照考核要求开发一个“乡村振兴数据可视化”大屏项目。请执行以下初始化任务：

使用 Vue 3 (Vite) + JavaScript 搭建项目。

安装并配置 vue-router（包含首页、产业分析、生态环境三个路由）。

安装 echarts，并创建一个名为 ECharts.vue 的通用基础组件，支持自动调整大小（Resize）和 Props 传参。

设计符合“乡村振兴”主题的深蓝色科技感 CSS 全局变量（如：背景色 #051932，主色调 #00f2fe）。

目录结构需规范：src/api（存放数据请求）、src/views（页面）、src/components（图表组件）、src/assets/json（静态数据）。

第二阶段：数据结构设计（核心数据支撑）
目标： 满足要求 5（静态 JSON）和要求 8（原创规范）。

提示词：
请在 src/assets/json/ 目录下创建一个 data.json 文件，模拟真实的乡村振兴业务数据。数据结构需包含以下 6 类：

农产品产量趋势（近 10 年，用于折线图）。

乡村产业占比（如：种植业、畜牧业、旅游业，用于饼图）。

各行政村人口密度（用于地图或柱状图）。

村民满意度评价（5 个维度，用于雷达图）。

实时环境指标（温度、湿度、PM2.5，用于仪表盘）。

特色非遗项目列表（包含名称、分类、传承人数量，用于列表展示）。
请确保数据逻辑合理，字段命名清晰。

第三阶段：核心页面与图表开发（功能与交互）
目标： 满足要求 2（3 个页面/6 个子页面）、要求 3（6 种 ECharts 图表）和要求 6（交互反馈）。

提示词：
请帮我完成 HomeView.vue（大屏主页）的开发：

布局：采用典型的“左-中-右”三栏布局，中间放置 3D 或 2D 效果的乡村地图，两侧各放置 3 个 ECharts 图表。

图表类型：必须实现折线图、柱状图、饼图、雷达图、仪表盘和热力地图。

交互增强：

图表需具备 Tooltip 悬浮提示。

实现“图表联动”：点击地图上的某个区域，侧边栏的柱状图自动切换为该区域的数据。

增加动态数字翻牌器显示“乡村年总产值”。

视觉优化：所有图表需使用统一的渐变色系，体现科技感，组件切换需有平滑过渡动画。

第四阶段：主题美化与响应式（视觉与风格）
目标： 满足要求 4（非遗主题特色、布局合理）。

提示词：
现在的项目需要更深层次的主题融合：

非遗特色：在 Header 标题栏两侧加入“中国结”或“祥云”纹理的 SVG 装饰，字体使用具有文化底蕴的样式。

响应式适配：使用 CSS Transform (scale) 或 Flexbox 确保页面在 1920x1080 和 1366x768 屏幕下均能完美平铺，不产生滚动条。

反馈优化：为所有可点击的图表项添加 cursor: pointer 样式，并确保跳转页面（如产业分析页）时有 Loading 动画。
