- 路由页面
  - 提供首页 `Home`，产业分析 `Industry`，生态环境 `Ecology`（共 3 个一级页面）
  - 产业分析与生态环境均做了主页面+子页面结构（产业 3 个子页：总览/产业链进度/非遗产业化；生态 3 个子页：实时监测/生态保护/治理执行）

- 首页大屏功能（`HomeView.vue`）
  - 左-中-右三栏布局
  - 首页中部“乡村地图（点击联动）”：点击地图节点联动右侧“行政村人口密度”柱状图高亮与更新
  - 顶部/核心位置展示“乡村年总产值”动态数字翻牌（`CountFlipper.vue`）

- 图表覆盖（ECharts，且每个图表支持 tooltip）
  - 折线图：近 10 年农产品产量趋势
  - 柱状图：行政村人口密度（与地图点击联动）
  - 饼图：乡村产业占比
  - 雷达图：村民满意度 5 维度
  - 仪表盘：实时环境仪表盘（温度）
  - 热力图：农村活跃度热力图（周维度强度展示）

- 数据来源
  - 所有可视化数据来自本地静态 JSON：`src/assets/json/data.json`
  - 提供产业、生态、治理、预警、非遗等多维数据结构，支持页面图表渲染

- 交互与稳定性
  - 图表组件封装为通用 `src/components/ECharts.vue`：
    - 支持 `option` 传参
    - 自动监听窗口 `resize`
    - 组件卸载时 `dispose`，避免内存泄漏
  - 首页对“数据就绪”做了保护，确保切换/加载阶段不会触发 ECharts 初始化异常

- 视觉与适配
  - 深蓝科技感主题（背景 `#051932`、主色 `#00f2fe`），卡片化布局与渐变风格统一
  - 全局样式限制滚动条，按 1920x1080 与 1366x768 的可视区完成自适应展示
